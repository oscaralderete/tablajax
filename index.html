Tablajax Demo

This table gets a practical submenu for custom actions driven by JavaScript, you can call URLs too, but that's not the Ajax way. Continent and Population columns are sortables.

All of the actions markup is generated by the SQL query as you can see in the detail below of each table. There are an extra configuration, but please check the repo file: /tablajax/countries.php for more details. 

			
SELECT
	Code,
	Name,
	Continent,
	Population,
	CONCAT("<a data-name=\"", Name, "\" data-id=\"", Code, "\" onclick=\"App.tryDetails(this)\">Check details</a>") AS act01,
	CONCAT("<a data-name=\"", Name, "\" data-id=\"", Code, "\" onclick=\"App.tryEdit(this)\">Edit record</a>") AS act02,
	CONCAT("<a data-name=\"", Name, "\" data-code=\"", Code, "\" onclick=\"App.tryDelete(this)\">Delete record</a>") AS act03
FROM
	country
WHERE
	Continent LIKE "%America"
ORDER BY
	Name ASC

In the example below the columns: Country code, District and Population are sortables.

There is another example here: second-demo.html with some more details. 

			
SELECT
	CountryCode,
	Name,
	District,
	Population,
	CONCAT("<button data-name=\"", Name, "\" data-id=\"", ID, "\" onclick=\"App.tryEdit(this)\">Edit</button>") AS act01,
	CONCAT("<button data-name=\"", Name, "\" data-id=\"", ID, "\" onclick=\"App.tryDelete(this)\">Delete</button>") AS act02
FROM
	city
WHERE
		1
ORDER BY
	Name ASC
